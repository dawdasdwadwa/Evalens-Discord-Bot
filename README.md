# Wildsync Bot — Приветствие и верификация

Discord-бот с двумя функциями:

1. **Приветствие** — при заходе нового участника генерируется картинка
   (аватар, ник, номер участника, название сервера) и отправляется
   в канал приветствия.
2. **Верификация по кнопке** — в канале верификации публикуется embed
   с кнопкой; при нажатии участнику выдаётся роль.

## Структура проекта

```
wildsync-bot/
├── assets/fonts/        # шрифты Poppins для карточки приветствия
├── cogs/
│   ├── welcome.py       # логика приветствия
│   └── verification.py  # логика верификации
├── config/
│   └── settings.py      # чтение переменных окружения
├── utils/
│   └── image_generator.py  # генерация PNG-карточки (Pillow)
├── bot.py               # точка входа
├── requirements.txt
└── .env.example
```

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Скопируйте `.env.example` в `.env`:
   ```
   cp .env.example .env
   ```

3. Заполните `.env`:
   - `DISCORD_TOKEN` — токен вашего бота (Discord Developer Portal → Bot → Token)
   - `WELCOME_CHANNEL_ID` и `VERIFICATION_CHANNEL_ID` — уже заполнены вашими ID
   - `VERIFIED_ROLE_ID` — **обязательно** укажите ID роли, которая будет
     выдаваться после верификации (ПКМ по роли → Copy ID, нужен включённый
     режим разработчика в Discord)
   - `UNVERIFIED_ROLE_ID` — если у вас есть отдельная роль "новичок/неверифицирован",
     укажите её ID, она будет автоматически сниматься после верификации

## Права бота (Developer Portal → Bot)

Включите **Privileged Gateway Intents**:
- `SERVER MEMBERS INTENT` (обязательно, иначе не будет работать приветствие)

Права бота на сервере (роль бота должна быть **выше** роли, которую он выдаёт):
- `Manage Roles`
- `Send Messages`, `Embed Links`, `Attach Files`
- `Use Application Commands`

## Запуск

```
python bot.py
```

## Настройка панели верификации

После запуска бота зайдите в канал верификации и один раз выполните
слэш-команду (доступна только администраторам):

```
/setup_verification
```

Бот опубликует embed с кнопкой. Кнопка постоянная — работает даже после
перезапуска бота, повторно вызывать команду не нужно (если только вы не
хотите продублировать панель).

## Как выглядит приветствие

Карточка генерируется автоматически в стиле:
- бейдж "Member #N" сверху
- круглый аватар пользователя в центре
- "Welcome {никнейм}"
- "to"
- "#название_сервера"

Цвета и вёрстку карточки можно изменить в `utils/image_generator.py`
(константы `COLOR_CREAM`, `COLOR_TEAL_LIGHT`, `COLOR_TEAL_DARK` и др.).

## Деплой на Railway

Проект готов к деплою на Railway так же, как ваш TalentShopBot:
1. Подключите репозиторий к Railway
2. Добавьте переменные окружения из `.env` в настройках проекта
3. Railway автоматически определит `requirements.txt` и запустит `python bot.py`

Не забудьте добавить `Procfile` или задать Start Command `python bot.py`
в настройках Railway, если это потребуется.
